CSS Stache

SCSS, Javascript, Ruby on Rails

CSS Stache is a Rails app that showcases CSS code snippets. Visitors can use the code, show off their own contributions, and play with the code using an in-browser text editor.

InTune

Ruby on Rails, HTML, CSS,
Last.fm API

InTune compares Last.fm users with their friends to see who's on the same wavelength.

The app collects user data from the Last.fm API, and sorts the user's friends by most shared artists. A simple bar chart allows users to see which artists they have in common with each friend.
